# QuizApp
Quiz app is an application where user can play and test his/her knowledge.The game has rounds, leader board, rules and many more.
The topics covered in this application are listed as below :
1. Sports.
2. Educational.
3. Bussiness.
4. Entertainment.
5. General Knowledge.
